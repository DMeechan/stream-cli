`stream debug`
==============

debugging tools

* [`stream debug:token`](#stream-debugtoken)

## `stream debug:token`

```
USAGE
  $ stream debug:token

OPTIONS
  -j, --jwt=jwt  (required) The JWT token you are trying to debug.
  -r, --raw      A raw object containing the data from your JWT.
```
